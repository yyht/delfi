delfi is a Python package for density estimation likelihood-free inference.

Warning

The code in this repository is still experimental, and APIs are subject to change without warning.

You can install delfi from PyPI using:

pip install delfi --user --process-dependency-links

For more details on the installation, see Installation.

Getting started describes basic usage of delfi.
